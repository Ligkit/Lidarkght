# Lidarkght
***

## 游戏介绍

Lidarkght是由Lit Studio团队开发的一款开源小游戏, 由基于Easy2D(V2.0.28)游戏引擎的C++语言编写而成.

本游戏分为`13`个场景.

![1](https://github.com/LitStudio-Center/Lidarkght/blob/main/README/img/1.png)

## Tip

游戏中有很多隐藏彩蛋, 全部发现有一定难度. 
每个彩蛋展示出来后, 窗口左上角会出现`获得成就: XXX`字样, 并逐渐淡出.
#### 所有成就: 
1. 撞墙
2. 你发现了黑板
3. 扫雷
4. 窗外的景色也不错
5. 楼外有楼
6. 还是这里
7. 你发现了钢琴
8. 又是这里
9. 柳暗花明又一村
10. 结束了
